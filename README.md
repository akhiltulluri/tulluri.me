# My Website

Powered by [zola](https://getzole.org/) and [serene theme](https://github.com/isunjn/serene).

## Install zola

[Guide](https://www.getzola.org/documentation/getting-started/installation/)

```zsh
brew install zola
```

## Build the site

```zsh
zola build
```

## Serve

```zsh
zola serve
```
